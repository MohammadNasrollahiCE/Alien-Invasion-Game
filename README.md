# Alien-Invasion-Game
This project, titled "Alien Invasion," is a graphical game developed using Python , Pygame library and the Anaconda Python distribution  .
.
.
In this game, aliens have invaded, and the user controls a spaceship that can
move horizontally and fire at them. The aliens are programmed to move both horizontally
and vertically; specifically, they shift down by one unit each time they traverse the width of the screen.
If the aliens collide with the spaceship or the ground, the user loses a life and the game
freezes for two seconds; after three consecutive losses, the game ends ("Game Over").
.
.
The alien ships are programmed to stay within the screen boundaries, and the game
is designed to be compatible with any screen size in full-screen mode.
